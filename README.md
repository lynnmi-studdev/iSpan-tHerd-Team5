# FlexWebSite
# 電商購物網站 - 訂單模組開發

> 此專案為資展國際（前資策會）全端工程師培訓團隊專案  
> **本人負責：訂單模組（ORD）完整開發**

📦 我的職責範圍（ORD 模組）

### 前台功能
- **購物車系統**：商品加入、數量調整、優惠券（自動計算最優惠券與更優惠券）、運費門檻邏輯、訂單金額即時計算
- **綠界金流 API 串接**：第三方支付流程、交易狀態處理
- **會員訂單查詢**：訂單歷史、即時物流追蹤
- **7 天鑑賞期退換貨申請**：符合台灣消保法規定的退換貨流程

### 後台功能
- **訂單查詢與狀態管理**：多條件查詢、訂單狀態流程控管
- **批量操作功能**：批量列印出貨標單、批量修改訂單狀態
- **RMA 售後系統**：退貨補寄審核、申請單狀態完整追蹤管理

**前端**

**第三方整合**

🎯 開發亮點

### 1. 金流串接實戰
成功整合綠界第三方支付系統，處理加密參數、交易回傳、錯誤重試機制

### 2. 批量操作優化
基於 3 年電商實務經驗，設計批量操作功能大幅提升後台管理效率

### 3. 完整業務邏輯
將真實電商營運流程（訂單、退換貨、庫存連動）轉化為系統功能

### 4. RMA 流程管理
設計完整的退換貨處理流程，包含多狀態追蹤與審核機制

📂 專案結構（ORD 訂單模組）

### 後端架構
```
tHerdBackend.Module/
└── tHerdBackend.ORD.Rcl/          # 訂單模組主體

tHerdBackend.Core/
└── DTOs/ORD/                      # 訂單資料傳輸物件

tHerdBackend.Infra/
└── Repository/ORD/                # 訂單資料存取層

tHerdBackend.SharedApi/
└── Controllers/Module/ORD/        # 訂單 API 控制器
```

### 前端架構
```
tHerdFrontend/src/
├── pages/modules/ord/             # 訂單頁面
│   ├── Cart.vue                  # 購物車
│   ├── Checkout.vue              # 結帳頁面
│   └── Orders.vue                # 訂單查詢
└── router/ord.js                  # 訂單路由配置
```

💡 從業務到技術

**3 年電商實務經驗轉化為技術解決方案：**
- 痛點：批量列印標單要一張張複製貼上 → 開發批量操作功能
- 痛點：訂單狀態要逐筆手動修改 → 設計批量狀態管理
- 痛點：庫存警示靠人工監控 → 建立系統自動化流程
